ruby on rails tutorial学习之github常用命令

github使用时常用的一些命令

在开发rails的过程中，经常会需要用到github管理自己的代码，学习git的一些常用命令，可以让我们更加高效地进行开发。

1.首次安装git需要进行的系统设置

git config –global user.name “your username”

git config –global user.email “your email”

2.然后运行git init进行仓库初始化，git默认跟踪所有文件的变化，但是在根目录下的.gitignore文件中记录的是git不需要跟踪的文件和目录。

3.git的主要命令：

  • git add . #将当前目录及其子目录下，所有改动且未提交的文件添加到git暂存区域中

  • git status #查看暂存区有哪些文件

  • git commit -m “your comment” #告知git保存改动（本地更改）

  • git push #将更改提交到远程仓库

  • git log #查看提交的历史信息

  • git checkout -f #切换到上一个提交记录来撤销本次改动

  • git remote add origin <远程仓库地址> #告知git添加仓库地址为代码的原本，代表本地的主分支

  • git push -u origin master #将本地仓库推送到github上

  • git checkout -b #新增一个分支，并切换到这个分支

  • git branch #查看所有分支

  • git checkout master #切换到主分支

  • git merge #将这个分支的改动合并到主分支

  • git branch -d #将这个分支删除

  • git checkout -D #放弃对从分支做的修改

  • git push #将改动推送到github
